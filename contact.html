⚙️ Developer Setup – Form Integration

To activate email delivery and GitHub data saving, complete these one-time setup steps. Replace the placeholder values in main.js.

1
EmailJS Setup: Go to emailjs.com → Create free account → Add Gmail service → Create a template with variables: {{from_name}}, {{from_email}}, {{from_phone}}, {{enquiry_type}}, {{message}} → Copy your Public Key, Service ID, Template ID into main.js.
2
GitHub Gist Setup: Go to gist.github.com → Create a new public or secret gist named enquiries.json with content [] → Copy the Gist ID from the URL → Create a Personal Access Token at GitHub Settings → Developer Settings → Token (classic) with gist scope → Add both to main.js.
3
Replace placeholders in main.js: YOUR_EMAILJS_PUBLIC_KEY, YOUR_EMAILJS_SERVICE_ID, YOUR_EMAILJS_TEMPLATE_ID, YOUR_GITHUB_TOKEN, YOUR_GIST_ID. The recipient email is already set to pusukuri.ramarao.75@gmail.com.
4
Deploy: Upload all 5 files (index.html, about.html, products.html, contact.html, style.css, main.js, logo.jpeg) to your hosting. Every enquiry will appear in your Gist JSON and an email will be sent to your Gmail address.
